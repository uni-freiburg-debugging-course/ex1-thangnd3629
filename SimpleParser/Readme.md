# Readme

## Enviroment
1. Java JDK 17
2. Maven 3.7 ( Or use maven wrapper included in source code)

## Parser
1. Build parser using maven command
```
mvn clean package 

or

./mvnw clean package
```
2. Run parser and evaluate expression
```
java -jar target/SimpleParser-v1-jar-with-dependencies.jar
```
